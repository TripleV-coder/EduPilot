# Documentation EduPilot

## Structure

```
docs/
├── architecture/      # Documentation de l'architecture
├── implementation/    # Plans et suivi d'implémentation
├── guides/            # Guides et solutions
├── sessions/          # Résumés de sessions
├── reports/           # Rapports de statut
├── INDEX.md          # Index principal
└── *.md              # Guides supplémentaires (deployment, security, etc.)
```

## Catégories

### Architecture (`docs/architecture/`)
- Vue d'ensemble de la plateforme
- Architecture frontend/backend
- Analyses détaillées

### Implementation (`docs/implementation/`)
- Plans d'implémentation
- Suivi des phases
- Jalons complétés

### Guides (`docs/guides/`)
- Guides de déploiement
- Solutions aux problèmes
- Améliorations

### Sessions (`docs/sessions/`)
- Résumés de sessions de travail
- Rapports finaux

### Rapports (`docs/reports/`)
- Statuts de progression
- Rapports de complétion

## Fichiers racine

| Fichier | Description |
|---------|-------------|
| `README.md` | Documentation principale du projet |
| `INDEX.md` | Index de la documentation |
| `DEPLOYMENT_GUIDE.md` | Guide de déploiement |
| `SECURITY.md` | Documentation sécurité |
| `TROUBLESHOOTING.md` | Dépannage |
| `QUICK_FIX_LOGIN.md` | Correction rapide login |
